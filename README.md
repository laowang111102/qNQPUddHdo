# 项目概述 python110

本项目是一个基于 Python 和 Django 开发的图书管理系统，旨在为用户提供便捷的图书借阅和管理服务。系统使用 SQLite 数据库进行数据存储。

## 技术栈

- **后端**: Python, Django
- **数据库**: SQLite

## 功能模块

### 基础功能

- 登录模块
- 图书管理
- 借阅管理
- 个人管理

### 高级功能

- 会员管理
- 数据可视化图表

### 核心亮点

- **续借功能**: 用户可以方便地进行图书续借操作。
